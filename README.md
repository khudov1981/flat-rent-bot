# Календарь бронирования для посуточной аренды

Это приложение представляет собой календарь бронирования для управления посуточной арендой жилья. Приложение реализовано как Telegram Mini App и позволяет пользователям управлять объектами размещения и бронировать даты.

## Функционал

### Управление объектами размещения
- Добавление новых объектов размещения
- Редактирование информации об объектах
- Удаление объектов размещения
- Просмотр списка бронирований для каждого объекта
- Выбор объекта для перехода к календарю бронирования

### Календарь бронирования
- Интерактивный календарь с возможностью выбора дат
- Визуальное отображение забронированных дат
- Навигация между месяцами
- Отображение выбранных дат с возможностью бронирования
- Подсчет стоимости бронирования
- Ручное бронирование с внесением данных о клиенте (ФИО и телефон)

### Уведомления администратора
- Отправка уведомлений о новых бронированиях через Telegram бота
- Подробная информация о клиенте и бронировании в уведомлении

### Навигация по страницам
- Отдельные страницы для управления объектами и календаря
- Удобная навигация между разделами

## Установка и запуск

1. Установите зависимости:
   ```bash
   npm install
   ```

2. Настройте переменные окружения:
   - Скопируйте файл `.env.example` в `.env`
   - Заполните переменные окружения для работы с Telegram ботом (см. раздел "Настройка уведомлений")

3. Запустите приложение в режиме разработки:
   ```bash
   npm run dev
   ```

4. Для сборки production-версии:
   ```bash
   npm run build
   ```

## Настройка уведомлений через Telegram бота

Для работы функции уведомлений администратора необходимо настроить Telegram бота:

1. Создайте бота через @BotFather в Telegram:
   - Напишите `/newbot` и следуйте инструкциям
   - Сохраните полученный токен бота

2. Получите ID чата администратора:
   - Напишите своему боту любое сообщение
   - Перейдите по ссылке: `https://api.telegram.org/bot<YOUR_BOT_TOKEN>/getUpdates`
   - Найдите свой chat_id в ответе

3. Настройте переменные окружения:
   - Скопируйте файл `.env.example` в `.env`
   - Заполните переменные:
     ```
     VITE_BOT_TOKEN=ваш_токен_бота
     VITE_ADMIN_CHAT_ID=ваш_chat_id
     ```

4. Перезапустите приложение

## Использование

### Навигация
Приложение имеет две основные страницы:
1. **Объекты размещения** - для управления объектами аренды
2. **Календарь бронирования** - для бронирования дат

Переключение между страницами осуществляется через навигационное меню в верхней части экрана.

### Управление объектами размещения
1. Нажмите кнопку "Добавить объект" для создания нового объекта размещения
2. Заполните форму с названием, описанием, адресом и ценой за ночь
3. Для редактирования объекта нажмите кнопку с карандашом на карточке объекта
4. Для удаления объекта нажмите кнопку с корзиной на карточке объекта
5. Для перехода к бронированию выберите объект и нажмите "Выбрать для бронирования"
6. В карточке объекта отображается список бронирований с информацией о клиентах

### Бронирование дат
1. Перейдите на страницу "Календарь бронирования" (автоматически открывается после выбора объекта)
2. В календаре выберите доступные даты для бронирования (забронированные даты отмечены красным цветом)
3. Просмотрите информацию о выбранных датах и общей стоимости
4. Нажмите кнопку "Ввести данные клиента" для перехода к форме бронирования
5. Введите ФИО клиента и номер телефона
6. Подтвердите бронирование

После подтверждения бронирования администратор получит уведомление через Telegram с информацией о клиенте и бронировании.

## Хранение данных

Все данные (объекты размещения и бронирования) хранятся в localStorage браузера, что позволяет сохранять информацию между сессиями.

## Технические детали

- Приложение автоматически определяет пользователя Telegram и отображает персонализированное приветствие
- Все данные сохраняются локально в браузере пользователя
- Приложение адаптировано для мобильных устройств
- Данные о бронированиях включают информацию о клиентах (ФИО и телефон)
